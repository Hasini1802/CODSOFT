# CODSOFT Machine Learning Internship Projects

## Overview

This repository contains the Machine Learning projects completed as part of the CodSoft Machine Learning Internship Program.

The projects demonstrate the application of Machine Learning, Data Analysis, Natural Language Processing (NLP), and Classification techniques using Python and Scikit-Learn.

---

## Task 1: Movie Genre Classification

### Objective

Build a machine learning model that predicts the genre of a movie based on its plot summary or textual description.

### Technologies Used

* Python
* Pandas
* Scikit-Learn
* TF-IDF Vectorization
* Naive Bayes Classifier

### Features

* Text preprocessing
* Feature extraction using TF-IDF
* Genre prediction from movie descriptions
* Model evaluation using accuracy metrics

---
